Bühnenaufbau
============

Übersicht
---------

Sämtliche Schweinwerfer und die Rauchmaschine sind an einem DMX512-Bus 
angeschlossen. Der Bus wird über einen Computer, auf welchem grandMA2 onPC 
läuft gesteuert.
